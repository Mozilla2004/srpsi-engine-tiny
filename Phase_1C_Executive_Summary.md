# Phase 1C 零样本外推测试 - 执行摘要

**测试目标**: 验证模型在 4 倍训练长度（T=200）的预测能力

**核心指标**: 外推比率 = Long-term Error / Short-term Error

---

## 🏆 结果排名

| 排名 | 模型 | 外推比率 | 核心发现 |
|------|------|---------|---------|
| 🥇 | **Transformer** | **0.24x** | 误差在外推区域下降 76%！ |
| 🥈 | **SRΨ Real** | **0.41x** | R 算子提升 18% |
| 🥉 | **SRΨ w/o R** | **0.50x** | 中等表现 |
| ❌ | **Conv** | **1.02x** | 无改善，虚假胜利 |

---

## 🎯 关键发现

### **1. Transformer 的惊人表现** 🤯

```
Short-term Error:  1.263
Long-term Error:   0.301
Extrapolation Ratio: 0.24x
```

**这意味着什么？**
- 在外推区域（48-200 步）的误差 **只有** 训练区域的 **24%**
- **误差在外推过程中下降了 76%！**
- Transformer 学到了真实的物理规律，不是拟合！

**物理学解释**：
- 自注意力机制在每个预测步骤都能看到整个历史
- 不像 RNN 那样逐步传递信息
- 能够维持能量和动量的长期稳定性

---

### **2. Conv 的平庸表现** ❌

```
Extrapolation Ratio: 1.02x
```

**这意味着什么？**
- 外推误差 ≈ 训练误差，无改善
- Conv 只是记忆了训练数据的模式
- **低 Loss (44.26) 确实是"虚假胜利"** ✅

---

### **3. SRΨ 的中等表现** 🥈

```
SRΨ Real:     0.41x
SRΨ w/o R:    0.50x
```

**这意味着什么？**
- R 算子提升 18%（0.41 vs 0.50）
- 优于 Conv，但不如 Transformer
- **SRΨ 是空间不变性之王，Transformer 是守恒性与外推能力之王**
